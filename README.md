# 📐 Quantity Measurement Application – Frontend  
## Clean UI for Measurement Conversion, Comparison, and Operations

---

## 📌 Project Overview

The **Quantity Measurement Application – Frontend** is designed to provide a simple, structured, and interactive user interface for performing quantity-related operations.

This project focuses on building the frontend of the application in two different ways while keeping the overall project idea the same:

- **Vanilla HTML, CSS, and JavaScript**
- **Angular with TypeScript**

The aim is to maintain a clean UI structure, organized codebase, and scalable frontend architecture.

---

## 📂 Branch Structure

This repository contains multiple branches for different frontend implementations of the same application.

- `main` – stable branch  
- `dev` – development branch  
- `feature/frontend-vanilla-html-css-js` – frontend using core web technologies  
- `feature/frontend-angular` – frontend using Angular  

---

---

## 🌐 Vanilla Frontend Branch

---

### `feature/frontend-vanilla-html-css-js`

This branch contains the frontend implementation built using **HTML5, CSS3, and JavaScript**.

The purpose of this branch is to develop the application using core web technologies with a simple and direct frontend structure.

### Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- JSON Server  

### Structure

```text
frontend/
│
├── html/
│	└── index.html
├── css/
│   └── styles.css
├── js/
│   ├── app.js
│   ├── api.js
│   └── utils.js
└── data/
    └── db.json
```

## ⚡ Angular Frontend Branch
